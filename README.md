# 🛡️ Nexus Antivirus

**AI-Powered Protection • Real-Time Security • Cloud Intelligence**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20Linux%20%7C%20macOS-lightgrey)]()

Nexus Antivirus is a **modern, multi‑threaded antivirus application** built with Python and Tkinter. It combines **signature‑based detection**, **heuristic analysis**, **cloud intelligence** (VirusTotal), and an optional **AI assistant** to deliver comprehensive protection for your system.

---

## ✨ Features

| Category | Feature |
|----------|---------|
| **🔍 Scanning** | Quick, Full, Custom, and Memory scans |
| **⚡ Performance** | Multi‑threaded scanning (8 threads) with real‑time progress |
| **🛡️ Protection** | Real‑time file monitoring, USB auto‑scan, ransomware shield (basic) |
| **📦 Quarantine** | Isolate threats with restore/delete options |
| **☁️ Cloud** | VirusTotal integration – scan with 70+ engines |
| **🤖 AI Assistant** | (Optional) ChatGPT‑powered threat explanations and security advice |
| **🔄 Updates** | Automatic virus definition updates via GitHub |
| **🚀 Startup** | Launch with Windows (Registry + Startup Folder) |
| **📊 Database** | SQLite for scan history, threat logs, user settings |
| **🎨 UI** | Modern dark theme with animated background and system tray icon |
